# Cyber Defense Architecture Design

## 🏗️ Overview

This architecture simulates a full cyber defense pipeline.

---

## 🔄 Flow

1. Logs are collected
2. IDS detects anomalies
3. SIEM correlates events
4. Threat Intelligence enriches data
5. SOAR triggers automated response

---

## 🛡️ Security Layers

* Detection layer (IDS)
* Analysis layer (SIEM)
* Intelligence layer (TIP)
* Response layer (SOAR)

---

## 🎯 Goal

Provide end-to-end visibility and automated defense.
